#### 原型链
```

### call,apply,bind的区别

<details>
<summary>展开查看答案</summary>

都是用来改变函数的this对象的指向的
```
let a = {
    name: 'a',
    age: 12,
    say: function(){
        return this.name + this.age;
    }
}
let b = {
    name: 'b',
    age: 20
}
a.say.call(b);
a.say.apply(b);
a.say.bind(b)();

a.say.call(b, 'new b', 10);
a.say.apply(b, ['new b', 10]);
a.say.bind(b, 'new b', 10)();
a.say.bind(b)('new b', 10);

```
> 注意一旦函数通过bind传递了有效的this对象，则该函数在运行期的this将指向这个对象，即使通过call或apply来试图改变this的指向也是徒劳的。
